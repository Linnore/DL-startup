# DL-startup
## Goal
Build up a workflow-notebook for NN training.

## To begin with 
1. Install anaconda or any other conda.
2. Create an empty and new environment by `conda create --name yourname python==3.9.12`.
3. Activate the environment you just created.
4. Run the following code in your terminal to prepare the environment for this project. 
`pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r requirements.txt`

## Run the example workflow
1. Run everything in Cifar10dataset.ipynb to prepare the dataset.
2. Check the exampleWorkflow.ipynb notebook.
